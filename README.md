# vdife

给公司前端写的 ngconsole 项目多语言资源辅助工具。

## Features

* 编辑 ngconsole 仓库中的 html 时，扩展可以执行 ”同步 localize“ 命令
* 编辑 ngconsole 仓库中的 html 时，扩展可以执行 ”新增/修改 localize“ 命令
* 编辑 ngconsole 仓库中的 html 时，扩展可以高亮显示[不]存在的 localize 属性
